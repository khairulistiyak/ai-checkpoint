# 📝 Drafts / R&D Notes

এই folder এ তুমি যেকোনো `.md` file রাখতে পারো — ideas, notes, research, brainstorming, যা ইচ্ছা।

## Rules:
- এই folder এর files **agent auto-execute করবে না** — শুধু reference হিসেবে পড়বে
- যেকোনো নামে file বানাও — `feature-idea.md`, `api-research.md`, `random-thought.md`
- Agent কে বললে এখান থেকে পড়ে কাজ করবে

## Examples:
```
plan/drafts/
├── README.md              ← এই file
├── new-feature-idea.md    ← তোমার idea notes
├── color-palette.md       ← design research
├── sql-format-spec.md     ← file format R&D
└── anything.md            ← যা ইচ্ছা!
```
